# QuizMaster - Interactive Quiz Application

**Overview**  
QuizMaster is a modern, web-based quiz application built using Django. It provides a dynamic and user-friendly platform for testing knowledge, learning through interactive quizzes, and tracking performance. The application supports multiple-choice and true/false question types, with secure login access and real-time results.

---

## Key Features

* **Secure Access:** Users must log in to attempt any quiz, ensuring controlled participation.    
* **Interactive Interface:** Users can navigate quizzes using **Next** and **Previous** buttons, select answers visually, and track progress through a progress bar and question indicators.  
* **Timed Quizzes:** Each quiz includes a timer, encouraging timely completion and simulating real testing conditions.  
* **Immediate Results:** Upon submission, users receive scores, performance messages, and can review their answers.  
* **Admin Management:** Administrators can create, edit, and manage quizzes, questions, and answers through an easy-to-use web interface.  
* **Modern Admin Interface:** Integration with Jazzmin provides a clean and professional admin dashboard.

---

## Quiz Rules

* Maximum of **4 options** and minimum of **2 options** per question.  
* Exactly **1 correct answer** is required per question.  
* Login is **mandatory** to attempt a quiz.  
 

---

## Sample Use Case

* **Quiz Type:** Multiple-choice, True/False  
* **Topic Example:** Python Programming Basics  
* **User Experience:**

  1. User login and selects a quiz.  
  2. Questions appear one at a time, with clearly visible options.  
  3. User navigates through questions using Next/Previous.  
  4. Answers are recorded and visually highlighted.  
  5. After completing the quiz, the user submits and immediately receives a score and feedback.
